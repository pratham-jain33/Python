# Learning Python

This small project contains my notes and code examples for learning Python basics.

## What it is
- Intended as an educational example for beginners practicing Python.

## Requirements
- Python 3.8 or newer

## Files
- `quizGame.py`: a quiz game with 10 questions
- `number_guessing.py`: a game where user tries to guess a random number
- `rock_paper_scissors.py`: a simple rock-paper-scissors game against the computer
